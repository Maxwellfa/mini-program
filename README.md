## 介绍

#### 小程序是什么

- 定义：小程序是一种全新的连接用户与服务的方式，它可以在微信内被便捷地获取和传播，同时具有出色的使用体验；
- 特点：无需安装、触手可及、用完即走、无需卸载；
- 英文：mini program。（注：应用、App 之类的词被苹果公司和谐）

#### 有趣的比喻

- App 就像原配，一年用不了几次；
- 服务号就像小三，每个月固定用几次；
- 小程序就像炮友，用完即走。

#### 运行环境

| 运行环境	      | 逻辑层	       | 渲染层         |
| --- | --- | --- |
| iOS	          | JavaScriptCore | WKWebView      |
| 安卓	          | X5 JSCore	   | X5浏览器       |
| 小程序开发者工具 | NWJS	       | Chrome WebView |
